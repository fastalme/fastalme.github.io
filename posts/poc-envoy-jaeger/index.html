Poc Envoy Jaeger

Mar 30, 2021
1 min read

Este fin de semana estuve jugando con Envoy, una herramienta bastante interesante que puede ser utilizada para muchos objetivos.

En este caso, utilizo Envoy como proxy reverso para unos servicios de tipo “hola mundo” que interactúan entre sí. Envoy le brinda a todo este despliegue la capacidad de generar información de trazabilidad distribuida (interceptando los requests entre estos servicios) y enviándola a un servidor centralizado Jaeger.

Así, casi cualquier servicio que se tenga construído y desplegado puede adoptar esta caracterísca, con una pequeña modificación del código fuente.

Terminé reescribiendo un ejemplo que encontré en otro repositorio, actualizando las versiones de los componentes y los archivos de configuración utilizados (usando la última versión del API de Envoy, v3).
